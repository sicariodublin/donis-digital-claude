## IMPLEMENTATION NOTES

1. **File structure:** create `/pt/index.html` mirroring `donis-digital.html`,
   or a `pt.html` at root — either works. `/pt/` is cleaner for SEO 
   (`fsteyerdigital.com/pt/`).

2. **Language switcher:** add to nav, both EN and PT versions:
   ```html
   <div class="lang-switch">
     <a href="/donis-digital.html" class="lang-en">EN</a>
     <span>/</span>
     <a href="/pt/" class="lang-pt">PT</a>
   </div>
   ```

3. **hreflang tags** — add to both pages' `<head>` for SEO:
   ```html
   <link rel="alternate" hreflang="en" href="https://fsteyerdigital.com/donis-digital.html"/>
   <link rel="alternate" hreflang="pt-BR" href="https://fsteyerdigital.com/pt/"/>
   ```

4. **Currency display:** prices shown in R$ on the PT page, € on the EN page.
   No automatic currency conversion — these are deliberately different,
   market-appropriate price points.

5. **WhatsApp button:** add a floating WhatsApp click-to-chat button on the
   PT version specifically — this is the default contact method in Brazil.
   ```html
   <a href="https://wa.me/353XXXXXXXXX?text=Ol%C3%A1!%20Vi%20seu%20site%20e%20queria%20saber%20mais."
      class="whatsapp-float" target="_blank" rel="noopener">
     💬
   </a>
   ```

6. **Demo sites:** keep as-is for now (Bella Roma etc. work visually regardless
   of language). Translate later if a Brazilian client specifically needs a
   Portuguese demo.

7. **Sitemap:** add the `/pt/` URL to `sitemap.xml` once built.
